# 豆腐牌风扇

**兽装专用智能风扇** 是一款为 furry 兽装穿戴者量身打造的个性化散热装备。它深度结合兽装使用场景中的特殊需求，在散热效率、操控便捷性和续航体验上进行了全方位创新设计。

## 设计理念

我们的设计紧紧围绕三个核心：**高效、便捷、持久**。
*   **高效**：采用小型化风嘴与优化流道，将风力集中输送至兽装嘴部区域，高效散热。
*   **便捷**：大尺寸按键与简化UI设计，确保即使佩戴厚重的兽爪也能轻松进行盲操作。
*   **持久**：独创的快速更换电池系统，打破了传统内置电池风扇的续航焦虑，让清凉永不中断。

## 产品亮点

*   **场景化风嘴设计**：小口径风嘴，完美适配大多数兽装头部内部空间，送风更精准。
*   **兽爪友好操控**：超大物理按键与清晰的屏幕反馈，操控体验无负担。
*   **无限续航可能**：支持电池热插拔更换。配备两节电池即可实现“一用一充”，持续运行。
*   **智能交互屏幕**：集成多功能设置菜单，包括风速、定时、系统设置等，一目了然。
*   **坚固轻量化机身**：通体采用3D打印技术制造，在保证结构强度的同时，实现了轻量化与个性化。
