# 豆腐脳ワークショップ

**獣装愛好家のために設計された獣装専用スマートファン**をご利用いただき、ありがとうございます。このガイドは、素早く爽やかな体験を始めるお手伝いをします。

## 開封してすぐお使いいただけます

1.  **バッテリーの取り付け**：同梱の18650リチウムイオンバッテリーモジュールをそのままセットしてください。
2.  **電源を入れる**：スイッチを下に倒すと、画面が点灯し、ファンが起動して自己診断を開始します。
3.  **基本操作**：電源投入後、**左ボタン**または**右ボタン**を短く押すと機能を選択できます。

## 主な機能一覧

*   **バッテリー交換**：バッテリー収納部はクイックドロープデザインを採用しており、工具不要で瞬時に交換可能です。予備バッテリーと併用すれば「無限の稼働時間」を実現できます。
*   **風量調整**：風量設定画面で、**左ボタン（弱）** または**右ボタン（強）** を押すと、直接風量レベルを調整できます。
*   **メニューへのアクセス**：メニュー画面で、**左ボタン**または**右ボタン**を押して機能を選択し、さらに詳細な設定を行うことができます。

獣頭（フェイス）を装着し、あなたのために特別に作られた爽やかさと便利さを思う存分お楽しみください！
