# 豆腐ブランド ファン

**獣装専用スマートファン**は、ファリー（獣装）着用者のために特別に設計されたパーソナライズされた冷却装備です。獣装使用シナリオにおける特殊なニーズに深く合わせ、冷却効率、操作の容易さ、バッテリー持続時間の体験において、全方向からの革新的な設計を行いました。

## デザイン理念

私たちの設計は、3つの核心である**効率性、利便性、持続性**を中心に展開しています。
*   **効率性**：小型化されたノズルと最適化された気流経路を採用し、風力を獣装の口元エリアに集中して送風し、効率的に冷却します。
*   **利便性**：大型のボタンと簡素化されたUIデザインにより、分厚い獣の手（フットパウ）を装着していても、簡単にブラインド操作（視認せずに操作）が可能です。
*   **持続性**：独自開発のバッテリー高速交換システムは、従来の内蔵バッテリーファンの持続時間に対する不安を取り除き、涼しさを途切れさせません。

## 製品のハイライト

*   **シナリオに合わせたノズル設計**：小口径ノズルは、ほとんどの獣装頭部内部のスペースに完璧に適合し、より正確な送風を実現します。
*   **獣の手（フットパウ）に優しい操作**：特大の物理ボタンと明確な画面フィードバックにより、負担のない操作体験を提供します。
*   **無限の稼働可能性**：バッテリーのホットスワップ（動作中交換）をサポート。2本のバッテリーを用意すれば「1本使用、1本充電」の循環で連続稼働が可能です。
*   **スマートインタラクティブスクリーン**：風量、タイマー、システム設定などを含む多機能設定メニューを集約し、一目で確認できます。
*   **頑丈で軽量なボディ**：ボディ全体を3Dプリント技術で製造。構造強度を確保しながら、軽量化と個性化を実現しています。
