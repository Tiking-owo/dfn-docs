# 豆腐脑工坊

欢迎使用专为兽装爱好者设计的 **兽装专用智能风扇**。本指南将帮助您快速开启清凉体验。

## 开箱即用

1.  **安装电池**：直接放入随附的18650锂电池组件即可
2.  **开机**：向下扳动开关，屏幕亮起，风扇开机自检启动。
3.  **基础使用**：开机后，短按 **左键** 或 **右键**可选择功能

## 核心功能一览

*   **更换电池**：电池仓采用快速抽拉设计，无需工具，可瞬间更换电池，搭配备用电池可实现“无限续航”。
*   **调整风速**：在风速界面，按 **左键（减）** 或 **右键（加）** 直接调整风速等级。
*   **访问菜单**：在菜单界面，按 **左键** 或 **右键**选择功能，可进行更多设置。
