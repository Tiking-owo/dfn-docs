# 豆腐腦工坊

歡迎使用專為獸裝愛好者設計的 **獸裝專用智能風扇**。本指南將幫助您快速開啟清涼體驗。

## 開箱即用

1.  **安裝電池**：直接放入隨附的18650鋰電池組件即可。
2.  **開機**：向下扳動開關，螢幕亮起，風扇開機自檢啟動。
3.  **基礎使用**：開機後，短按 **左鍵** 或 **右鍵** 可選擇功能。

## 核心功能一覽

*   **更換電池**：電池倉採用快速抽拉設計，無需工具，可瞬間更換電池，搭配備用電池可實現「無限續航」。
*   **調整風速**：在風速界面，按 **左鍵（減）** 或 **右鍵（加）** 直接調整風速等級。
*   **訪問選單**：在選單界面，按 **左鍵** 或 **右鍵** 選擇功能，可進行更多設定。

戴上獸頭，盡情享受為您量身打造的清涼與便捷吧！
