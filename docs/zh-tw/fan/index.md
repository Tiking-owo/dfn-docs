# 豆腐牌風扇

**獸裝專用智能風扇**是一款為 furry 獸裝穿戴者量身打造的個人化散熱裝備。它深度結合獸裝使用場景中的特殊需求，在散熱效率、操控便捷性和續航體驗上進行了全方位創新設計。

## 設計理念

我們的設計緊緊圍繞三個核心：**高效、便捷、持久**。
*   **高效**：採用小型化風嘴與優化流道，將風力集中輸送至獸裝嘴部區域，高效散熱。
*   **便捷**：大尺寸按鍵與簡化UI設計，確保即使佩戴厚重的獸爪也能輕鬆進行盲操作。
*   **持久**：獨創的快速更換電池系統，打破了傳統內建電池風扇的續航焦慮，讓清涼永不中斷。

## 產品亮點

*   **場景化風嘴設計**：小口徑風嘴，完美適配大多數獸裝頭部內部空間，送風更精準。
*   **獸爪友善操控**：超大實體按鍵與清晰的螢幕回饋，操控體驗無負擔。
*   **無限續航可能**：支援電池熱插拔更換。配備兩節電池即可實現「一用一充」，持續運行。
*   **智慧互動螢幕**：整合多功能設定選單，包括風速、定時、系統設定等，一目了然。
*   **堅固輕量化機身**：通體採用3D列印技術製造，在保證結構強度的同時，實現了輕量化與個人化。
